<h1 align="center">📊 Student Performance Visualizer</h1>
<p align="center">Python-Based Data Analysis & Visualization Project</p>

---

## 🧾 Overview

This project analyzes student performance data and converts it into meaningful insights using graphical visualizations.

It demonstrates how raw data can be processed, analyzed, and presented effectively using Python.

---

## ✨ Features

- Accepts student names and marks as input  
- Calculates total, average, highest, and lowest scores  
- Identifies the top-performing student  
- Generates line and bar charts  
- Displays structured output  

---

## 🛠️ Technologies Used

- Python  
- Matplotlib  

---

## 🧠 Concepts Applied

- Data handling using lists  
- Modular programming  
- Problem solving  
- Data analysis  
- Data visualization  

---

## ▶️ How to Run

1. Install Python  
2. Install dependencies:
   ```bash
   pip install matplotlib
   ```
3. Run:
   ```bash
   python student_visualizer.py
   ```
